## 📌 Overview

The **AI-Powered PDF Assistant** is a full-stack application that transforms static PDF documents into intelligent, interactive data sources.

It enables users to:
- Upload PDF files securely  
- Process documents using a Python FastAPI backend  
- Extract meaningful insights from content  
- Interact with documents in real-time  

---

## ⚡ Key Features

✨ Secure PDF upload system  
📁 Automatic file storage with unique naming  
🧠 AI-based document processing  
⚡ FastAPI-powered backend for high performance  
🔒 PDF validation with size limits  
🔄 Real-time processing workflow  
🌐 Seamless Node.js + Python integration

## 🔍 Detailed Features

### 📁 Secure Upload System
- Validates file type and size before upload  
- Uses Multer for efficient file handling  
- Prevents invalid or harmful file uploads  

### 🧠 AI Processing Engine
- Extracts text and meaningful insights from PDFs  
- Processes content using FastAPI backend  
- Designed for scalability and performance  

### 🔄 Real-Time Processing Flow
- Immediate response after upload  
- Smooth backend communication between Node.js and Python  
- Efficient request handling  

### 🌐 Full Stack Integration
- Node.js handles API and routing  
- Python handles AI processing  
- Frontend provides user interaction

## 🚀 Installation & Setup

```bash
# Clone Repository
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name

# Run Node.js Server
cd server
npm install
npm start

# Run Python FastAPI Server
cd ../python-server
pip install -r requirements.txt
uvicorn main:app --reload

# Run Frontend
cd ../client
npm install
npm start
```

## 📘 Usage

1. Upload a PDF document  
2. System validates and stores file  
3. FastAPI processes the content  
4. AI extracts useful insights  
5. Results are returned to the user
